# Tecnologias utilizadas

* Vue (2.6+)
* Vue Router
* Vuex
* Vuetify
* Vuelidate
* Apollo Client
* Apollo Cache InMemory
* ChartJS
* Moment
* RxJS

A aplicação consome a [API GraphQL](https://github.com/fabiosnascimento/vue-finances-back) desenvolvida no curso.
